# DevOps CI/CD Project - React Frontend + Node Backend (Microservices)

This project demonstrates an end-to-end CI/CD pipeline deploying a React frontend and Node.js backend to Kubernetes (EKS) using:
- Jenkins
- Docker
- Ansible
- Kubernetes (EKS)
- Terraform (skeleton)
- Docker Hub / ECR (user to configure)
- Optional Helm chart
